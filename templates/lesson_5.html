Лучше всего использовать функцию объявления, т.к ее можно вызвать из любой части кода.
Функции при загрузке страницы все поднимаются вверх, и функция объявления создается интерпретатором до объявления кода.
Функция выражения не может быть вызвана до своего создания. В таком случае поднимется только пустая перменная.
Функции нужно называть маленькими буквами и при этом глаголами

область видимости переменной

var result = 25; // Глобальная

function()
{

var result = 2; // Локальная ( действует только в области функции )

}

name(); - вызов функции напрямую

a.name(); - вызов функции как методы

еще можно как конструкторы и косвенно используя call() и apply();

var res = 25;

1 >> function(){ var res = 20; }
2 function(){ var res = 15; }
вызов alert(res); // 15 - в этой области видимости

alert(res); //25 - потмоу-что глобальная.

Елси функция не находит нужную переменную у себя в области, то она обращается к родительской функции и спрашивает ее, если и там не находит то родительская функция обращается к своей родительской функции и т.д пока не найдет нужную.

Глобальная переменная может быть переопределена внутри функции которая ее вызвала.
return - возвращает значение из функции.
