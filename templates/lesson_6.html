клиентский Javascript

клиентский js состоит из трех частей

1) DOM - document opject model ( объектная модель документа )
2) BOM - browser object model ( объектная модель браузера )
3) js - собственно сам javascript

Все эти элементы это части большого глобального объекта window
DOM - это html разметка, его глобальный объект (document)
BOM - это функционал и данные браузера

Загрузка программы на js выполняются в два этапа

1 - интерпретация
2 - ассинхронные запросы, клиентские события
window.onload = function(){} - функция сработает после загрузки всего объекта
document.getElementById("ID"); - обращение к элементу разметки по его идентификатору (id)
Location - объект браузера для манипуляций
Navigator
  • - объект содержащий данные браузера
  • - appName - название браузера
  • - appVersion - версия, 4 или 5 поколение
  • - userAgent - версия и дополнительные сведения
  • - platform - сведения об ОС и платформу браузера
  • - onLine - свойство определяет подключен ли браузер к сети
  • - geolocation - свойство местоположения
screen
  • - объект данных об экране
  • - width - ширина экрана
  • - height - высота экрана
  • - availWidth - доступная ширина
  • - availHeight - доступная высота

window.location.reload(); - указывает браузеру перезапуститьстраницу
window.location.replace("http://test.com"); - редирект с заменой страницы
window.assign("путь"); - аналог replace только с возможностью возврата на предыдущую страницу

window.history.back(); - перемещаемся на предыдущую страницу
window.hisotry.forward(); - перемещаемся на следующую страницу
window.hisotry.go(колличество); - перемещаемся на указанную страницу минуя колличество страниц указанных в аргументах ( если значение
отрицательное, то движемся назад)

window.setTimeout( function(){}, время в миллисекундах ); - выполняет функцию через указанное время единожды
window.setInterval( function(){}, время в миллисекундах ); - выполняет функцию каждый раз через указанный промежуток времени
window.clearInterval( идентификатор счетчика ( обычно переменная которая его вызывает)); - очистка интервалов

WINDOW можно не указывать при вызове!!!

Navigator.javaEnabled(); - проверка на возврат апплетов явы.
Navigator.cookiesEnabled(); - проверка на сохранение куков
btn.onclick = function(){}; - функция срабатывает при клике на объект, объект это переменная соержащая обращение к элементу разметки

хранилища данных ( web storage - API )

LocalStorage - содержит данные пользователя даже после перезагрузки браузера (аналог куки)
Хранилище чувствительно к домену, порту и протоколу!

setItem( "name", value ); - занести значение с именем в хранилище
getItem ( "name" ); - получить значение из хранилища
removeItem ( "name" ); - удалить значение из хранилища
локальное хранилище это объектный массив, и перебирать его нужно циклом for
for( var name in localStorage[name] ) // Перебираем все элементы name циклом
var value = localStorage[name]; // заносим значения в новую переменную
