массивы

Массивы это переменные имеющие несколько значений через запятую.

var a = [1,2,3,4,5]
Задаются в виде ЛИТЕРАЛА или КОНСТРУКТОРА.
ЛИТЕРАЛ - var arr = [];
КОНСТРУКТОР - var a = new Array(); - в скобках задаем колличество элементов ИЛИ перечисляем их как в литерале.
колличество элементов в массиве всегда на 1 больше. Т.е чтобы получить последний
обозначенный элемент нужно задавать длину массива минус 1 ( arr.length - 1 )
Каждый элемент массива имеет собственный индекс, отсчет всех индексов начинается с 0
у массивов есть свойства и методы
Свойства возвращают какие-то значения массивов. Синтаксис ( arr.length ) это вернет длину массива
Методы это действия над массивами ( arr.pop(); ) это удалит последний элемент массива и вернет его значение.
Обращение к массивам

arr[1] - обращение к элементу массива с индеском 1.
arr[] - обращение ко всему массиву, выведет весь массив.
arr[5] = "test" - переопределит элемент массива с индексом 5.
arr[arr.length] - обращение к самому последнему элементу массива, он пустой.(undefined)
arr[arr.length-1] - обращение к последнему обозначенному элементу массива.э
arr.length = 5; - задает статическую длинну массива. Если в массиве больше элементов
то он режется до 5 индекса. т.е до 6 элемента в массиве
Разреженный массив - массив имеющий пустые значения.

методы

arr.push(); - добавит элемент в конец массива.

arr.pop(); - удалит элемент с конца массива и вернет его значение.

arr.shift(); - удалит элемент из начала массива и вернет его значение.

arr.unshift(); - добавит элемент в начало массива и возвращает новую длину массива.

arr.slice(начало, конец); - копирует значения массива
в диапазоне от "начала"(включительно) и до конца(не включительно) если конец задать с минусом то значения
начнут считаться с конца.

arr.splice(начало, колличество); - вырезает значения из массива в заданном колличестве после "начала".

arr.indexOf(искомое); - возвращает первый элемент который совпадет по поиску.( поиске слева направо )
если такого нет то вернет (-1)

arr.lastIndexOf(искомое); - аналог indexOf только поис идет с конца массива, если такого нет то вернет (-1)
arr.forEach(); - цикл for для перебора массива в упрощенном синтаксисе.
