Обработка ошибок

Обработчик ошибок window.onerror - перехватывает все ошибки браузера

window.onerror = function(){} - функция вызываемая каждый раз когда есть ошибка в браузере
функция должна принимать три аргумента

1 - Сообщение описывающее ошибку
2 - Строка с url документа с кодом который вызвал ошибку
3 - Номер строк в документе вызвавший ошибку
Если обработчик вернет true - это означает что ошибка обработана и сообщение об ошибке не выведется

window.onerror = function( msg, url, line ) //Перехватываем ошибку
{
if( onerror.num++ < onerror.max )
{
alert("ОШИБКА: " + msg + "\n" + url + ":" + line); // Выведет если найдет ошибку
return true;
}
}

onerror.max = 3;
onerror.num = 0;
unixtime - время в мили
объект Data(); - возвращает полную дату.
Он имеет множество методов для получения отдельных данных ( дни, часы, минуты )

var time = new Data();

- time.getYear(); - получаем год
- time.getDay(); - получаем день
- time.getHours(); - получаем часы
- time.getMinutes(); - получаем минуты
- time.getSecondes(); - получаем секунды
- time.getTime(); - получаем unixtime

если нужно задать значение используем 'set' вместо 'get';

toString(); - переводит Date в строку
toTimeString(); - возвращает строку возвращающую дату в локальном часовом поясе
valueOf()??; - преобразует объект в его внутренний миллисекундный формат

исключения

это сигналы об ошибках или исключительных ситуациях

Исключения возникают если программа грузится долго или если оно вызвано инструкцией 'throw'

throw 'Выражение'

Интерпретатор возбуждает исключения используя экземпляр класса Error одного из его подклассов.

Error имеет два свойства это
- name - тип ошибки
- message - строка переданная конструкции

например if(x < 0) throw new Error('Не может быть отрицательным');
При возникновении исключения интерпретатор прерывает выполнение программы и переходит к ближайшему обработчику исключений.
Обработчик исключений это конструкция try/catch/finally
Непосредственно как обработчик тут выступает 'catch', если такого нет, программа проверяет сл. внешний блок, если обработчик так и не будет найден, программа выдаст ошибку.
